# Karry (zzuyu) Personal Homepage

Static personal homepage prepared for GitHub Pages.

## Deploy with GitHub Pages

1. Create a GitHub repository named `<your-github-username>.github.io`.
2. Upload everything in this `deploy` directory to the repository root.
3. Open repository `Settings` -> `Pages`.
4. Set source to `Deploy from a branch`, branch `main`, folder `/ (root)`.
5. Visit `https://<your-github-username>.github.io/` after Pages finishes deploying.
